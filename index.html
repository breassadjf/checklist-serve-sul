Conectar ao Banco de Dados
Configure o Supabase para armazenar dados e fotos em nuvem, acessível por todos os dispositivos.
1
Acesse supabase.com → crie conta gratuita → clique em New Project
2
No projeto criado, vá em SQL Editor e execute o script do arquivo setup.sql que acompanha este sistema
3
Vá em Settings → API e copie a Project URL e a chave anon/public
4
Cole abaixo e clique em Conectar
